About

Hi there!
I am currently a third-year undergraduate student at the University of California, San Diego (UCSD), pursuing a Bachelor of Science in Computer Science with a minor in Mathematics. Since joining UCSD, I have been actively involved in Computer Graphics research at the Center for Visual Computing, where I have been fortunate to be advised by Professors Ravi Ramamoorthi and Tzu-Mao Li.

My primary research interests include:
  • Real-time, inverse, and differentiable rendering
  • Languages and systems for graphics and heterogeneous computing
  • Interactive simulation of natural phenomena
  • Reinforcement learning and machine learning
  • Computer vision systems for robotics

Open Source Projects

Javelin
Robust Just-In-Time (JIT) compilation framework to effortlessly generate graphics shaders at runtime using advanced C++ overloading and template metaprogramming. This framework presents a portable dialect of C++ that closely resembles GLSL shader programming enhancing efficiency in graphics application development.
Kobra
A 3D rendering engine developed in C++ using the Vulkan API, designed specifically for prioritizing research and development. It enables users to allocate their time and efforts on experimentation rather than implementation, facilitating creative exploration in 3D graphics.
Zhetapi
Modern C++ library designed for machine learning and numerical analysis, offering intuitive usability. Zhetapi also features a dedicated Python-like scripting language to streamline access to its core library functionalities.
littlevk
Lightweight, ergonomic, single-header wrapper for jump-starting graphics and visualization applications using Vulkan. Leverages C++20 features to ensure a smoother user experience.
SMake
Simplified build system for quickly configuring, building, and running C++ projects. The user-friendly configuration allows easy access for new users.
metacpp
Exploratory project to evaluate Lisp programs at compile-time with C++ metaprogramming.
tuicpp
Wrapper library for ncurses that offers intuitive abstractions for common constructs such as text windows, prompts, and tables.
Murphy
Exploratory graphics project where curves are transformed to match any given silhouette. Effectively uses differentiable rasterization via precise edge sampling for fast convergence.
