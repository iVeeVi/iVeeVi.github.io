About

Greetings! I am a third-year undergraduate student at UC San Diego studying computer science with a minor in mathematics. Since joining the university, I have been a part of the Center for Visual Computing, where I have been fortunate to work with professors Ravi Ramamoorthi and Tzu-Mao Li.

My prominent research interests include:
  • Real-time, inverse, and differentiable rendering
  • Languages and systems for graphics and heterogeneous computing
  • Interactive simulation of natural phenoma
  • Reinforcement learning and machine learning
  • Computer vision systems for robotics

Experience

I previously interned at NVIDIA (Summer 2024) as a 3D Graphics Software Intern, where I worked on the Slang project. Here, I explored realistic applications of shader modules, dynamic shader linking, and link-time specialization. I worked with a wonderful team while I was managed by Brandon Mills.

Open Source Projects

Javelin
A graphics engine system tailored for shader programming. Provides a JIT framework to generate shaders at runtime, that is programmed in a near-identical manner to GLSL using C++ overloading and template metaprogramming.
Kobra
A 3D rendering engine written in C++ using the Vulkan API. It is designed with research development in mind, so that more time can be spent towards experimentation rather than implementation.
Zhetapi
A modern C++ machine learning and numerical analysis library with an emphasis on intuitive usage. Older version also provides a scripting language to easily interface with core functionality.
Littlevk
A light-weight, ergonomic, single-header wrapper for jump-starting graphics and visualization applications using Vulkan. Leverages C++20 features to promote smoother user experience.
